JagStudio.
A complete Assembler, BASIC and 'C' Development Suite for the Atari Jaguar.


Please see the docs folder for a quickstart guide and reference manual.

The media folder contains the relevant logos for JagStudio, Raptor and U235.

Example projects are in the projects folder under the appropriate language.

Any new projects you create will appear in the \projects\basic\ , \projects\c\ or \projects\asm\ folders.

Type 'build' without the quotes from a command prompt to see all options.
